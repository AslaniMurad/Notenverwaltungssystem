Grade Management System
    A digital management software that simplifies the recording, organizing, and evaluating of student grades.
    As part of a school project, this initiative was launched to provide teachers, students, and administrators with a modern and user-friendly grade overview.
Presentation
    With the grade management system, students, teachers, subjects, and exams can be centrally managed.
    Teachers have the opportunity to record, adjust, and weight grades.
    It is clearly possible for students to view their grades, exams, and average scores.
    User accounts, roles, and system data are managed by administrators.
    The goal is to simplify school management processes and increase transparency regarding performance evaluation.
Features
    Registration of users with different roles (Administrator, Teacher, Learner)
    Management of teachers, students, subjects, and classes
    Collection, editing, and deletion of grades
    Automated calculation of subject and overall average grades
    Weighting of performances (such as e.g. Exams, participation, classwork)
    Presentation of performance charts and grade trends
    Connection to the database
Used technologies
    Programming language: Node JS
    Database: SQLite
    User interface: XHTML
    Version control: Git & GitHub
    Authentication: OAuth 2.0
    Design tools: HTML, JavaScript, CSS
